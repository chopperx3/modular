Este proyecto tiene como objetivo desarrollar una herramienta para renovar documentos viejos escritos a mano. Utiliza un escáner de documentos o permite la carga de imágenes por parte del usuario para convertir los textos manuscritos en letra de molde. La herramienta facilita la digitalización y modernización de documentos antiguos.
